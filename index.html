프로젝트 깃허브 테스트

1. 코드를 작성한다 2. 터미널 창에 -> git add . (입력해준다) 3. git commit -m "first commit" (최종본 이름을 적어준다) 4. git push origin main (main=브랜치/브랜치에 코드를 푸쉬한다) 5. 깃 허브 파일에서 확인한 후 잘 들어가있으면 성공!

강사님 버전으로 커밋

1. 코드를 작성한다 2. 왼쪽메뉴 파일밑에 점 3개 이어진 그림 클릭 3. 커밋 할 메시지 입력 후 커밋버튼 눌려준다 4. 변경 내용 동기화 버튼이 나오면 눌려준다 5. 깃 허브 파일에서 확인한 후 잘 들어가있으면 성공!
